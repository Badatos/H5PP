Libraries

To use H5PP you need a set of files named "libraries" that contains the necessary to create different types of content.
You can find the official release of H5P, containing many content types, at this adress :

Editor

To create or edit a content you need to use the editor provided by H5P.
All content is identified by a title, set during creation, and a identifier (ID) created automatically after the creation.
Everybody can view a content, but only authenticated users can create and edit contents. Finally, only superuser can delete content.

Contents

Anytime, you can consult the list of created contents.
It is through this list that you can view the contents.
